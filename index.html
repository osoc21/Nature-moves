The project

Our goal is to build a map visualisation and editing tool to enrich and integrate the Natuurpuntâ€™s nature reserves data into the OSM database, making it both accessible, maintainable, and mobile-friendly.

Features

The main use of the map is finding nature reserves, information about them, and points of interest around them. To be able to find the above mentioned more easily a filter is added in which you can select what you want to see on the map. Another helpful feature is follow your geolocation, when you press the location button it will go to your location. This button has two states, it will either follow your location when moving, or not move anymore after pressing it.

The pdf

Another prominent feature we added is a pdf export. When pressing the download button a pdf will be downloaded of the current map you are seeing. Depending on which theme of Mapcomplete you are using you can create a customized pdf layout! Since this theme is Natuurpunt, a header specialized for Natuurpunt is added at the top.
